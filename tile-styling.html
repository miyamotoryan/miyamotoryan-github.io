Overview

In February 2021, I was put in touch with the owner of Tile Styling who was struggling to find someone with the ability to fix their eCommerce site. After visually inspecting the site and testing functionality, a full redesign and rebuilding of the eCommerce system was suggested, but the work was out of the client's budget.

The main issues that the client was having was the inability to edit the behavior of plugins and other core system functions and designs. Since the website was built using WordPress, all modifications were done via PHP filters and JavaScript. The WooCommerce eCommerce system was also rebuilt using a combination of 3rd party plugins and custom code to become an inquiry/quotation based model instead of an online store.
